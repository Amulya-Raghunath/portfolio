Malware Detection using ML & Deep Learning

Project Overview

Developed an ML & Deep Learning based malware detection system using Kaggle & CIC-MalMem2022 datasets. Benchmarked classical models vs deep neural networks for binary & multi-class classification.

Tech Stack

  • Python, Google Colab
  • Scikit-Learn, TensorFlow, XGBoost
  • Matplotlib, Seaborn

Key Achievements

  • ~100% accuracy in binary malware detection
  • ~99.98% accuracy in multi-class classification
  • Performed feature engineering + correlation analysis

Online Attendance System using Facial Recognition

Project Overview

Developed a computer-vision system leveraging OpenCV, Haar-Cascade face detection, and LBPH face recognition to authenticate users and record attendance in real time. Integrated Python GUI, MySQL database, and role-based login for students and faculty.

Tech Stack

  • Python
  • Tkinter, OpenCV, NumPy
  • LBPH (Local Binary Patterns Histograms), Haar Cascade
  • Database: MySQL

Key Achievements

  • Two way aythentication for faculties
  • Simple to implement
  • Enabled CSV + MySQL reporting, allowing admins to export, store, and audit attendance data.
  • Optimized webcam input pipeline for smooth real-time face capture and recognition.
